## ğŸš€ Getting Started Steps

### Step 1: Read Documentation
1. Start with [QUICKSTART.md](./QUICKSTART.md)
2. Then read [SETUP_GUIDE.md](./SETUP_GUIDE.md)
3. Reference [API_DOCUMENTATION.md](./API_DOCUMENTATION.md) as needed

### Step 2: Configure
1. Open `.env.local`
2. Set `MONGODB_URI` with your MongoDB connection
3. Set `JWT_SECRET` to a secure value

### Step 3: Run Project
```bash
npm run dev
```
Application opens at: http://localhost:3000

### Step 4: Test Features
- Sign up at `/auth/signup`
- Log in at `/auth/login`
- Browse videos
- Change language
